18.  **การนับจำนวนครั้งที่สมาชิกปรากฏ (Count Occurrences):**
    
    -   **โจทย์:** กำหนด Array `int nums[] = {1, 5, 2, 5, 3, 5, 4};` และค่าที่ต้องการนับ `int target = 5;` ให้พิมพ์จำนวนครั้งที่ `target` ปรากฏใน Array.
        
    -   **ตัวอย่าง:**
        
        ```
        Number of 5s: 3
        
        ```
