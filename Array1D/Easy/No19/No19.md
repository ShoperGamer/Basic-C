19.  **การคัดลอก Array:**
    
    -   **โจทย์:** ประกาศ Array `int source[] = {1, 2, 3, 4, 5};` และ Array `int destination[5];` ให้คัดลอกสมาชิกทั้งหมดจาก `source` ไปยัง `destination` แล้วพิมพ์ `destination`.
        
    -   **ตัวอย่าง:**
        
        ```
        Destination Array: 1 2 3 4 5
        
        ```
