20.  **การย้อนกลับลำดับของ Array (Reverse Array):**
    
    -   **โจทย์:** กำหนด Array `int arr[] = {1, 2, 3, 4, 5};` ให้เปลี่ยนลำดับของสมาชิกใน Array ให้เป็นตรงกันข้าม (เช่น `5, 4, 3, 2, 1`) โดยแก้ไขใน Array เดิม.
        
    -   **ตัวอย่าง:**
        
        ```
        Reversed Array: 5 4 3 2 1
        
        ```
