1.  **การประกาศและกำหนดค่าเริ่มต้น:**
    
    -   **โจทย์:** ประกาศ Array 2 มิติ ชนิด `int` ขนาด 2 แถว 3 คอลัมน์ (2x3) และกำหนดค่าเริ่มต้นเป็น:
        
        ```
        { {1, 2, 3}, {4, 5, 6} }
        
        ```
        
    -   **ตัวอย่าง:** (แสดงผลในรูปแบบเมทริกซ์)
        
        ```
        1 2 3
        4 5 6
        
        ```
