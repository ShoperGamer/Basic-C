4.  **การพิมพ์ค่าทั้งหมดใน Array (ด้วย Nested Loop):**
    
    -   **โจทย์:** ประกาศ Array 2 มิติ ชนิด `float` ขนาด 3 แถว 2 คอลัมน์ (3x2) และกำหนดค่าเป็น:
        
        ```
        { {1.1, 1.2}, {2.1, 2.2}, {3.1, 3.2} }
        
        ```
        
        จากนั้นใช้ `for` loop ซ้อนกันเพื่อพิมพ์ค่าทั้งหมด.
        
    -   **ตัวอย่าง:**
        
        ```
        1.1 1.2
        2.1 2.2
        3.1 3.2
        
        ```
