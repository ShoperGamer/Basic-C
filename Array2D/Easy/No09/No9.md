    -   **โจทย์:** กำหนด Array 2 มิติ `int sq_matrix[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};` ให้พิมพ์เฉพาะสมาชิกที่อยู่บนแนวทแยงหลัก (เมื่อ index แถวเท่ากับ index คอลัมน์).
        
    -   **ตัวอย่าง:**
        
