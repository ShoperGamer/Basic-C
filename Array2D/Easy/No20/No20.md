20.  **การคัดลอก Array 2 มิติ:**
    
    -   **โจทย์:** ประกาศ Array `int source[2][2] = {{1, 2}, {3, 4}};` และ Array `int destination[2][2];` ให้คัดลอกสมาชิกทั้งหมดจาก `source` ไปยัง `destination` แล้วพิมพ์ `destination`.
        
    -   **ตัวอย่าง:**
        
        ```
        Destination Array:
        1 2
        3 4
        
        ```
