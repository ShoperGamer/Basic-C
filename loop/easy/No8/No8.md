### ข้อ 8: นับตัวอักษร 'a'

เขียนโปรแกรมที่รับจำนวนครั้ง (`N`) ที่ผู้ใช้จะป้อนตัวอักษรเข้ามา จากนั้นให้รับตัวอักษรเหล่านั้นทีละตัว และนับว่ามีตัวอักษร 'a' (ตัวพิมพ์เล็ก) ทั้งหมดกี่ตัวที่ถูกป้อนเข้ามา

**ตัวอย่าง:**

```
Enter number of characters: 5
Enter character 1: h
Enter character 2: a
Enter character 3: l
Enter character 4: l
Enter character 5: o
Number of 'a' is = 1

```
